# 🎉 SVU Event Space Booking System

## 📌 Project Description
The SVU Event Space Booking System is a web-based application developed to manage event space reservations efficiently. 
It allows users to book venues, check availability, and manage event details in a structured way.

## 🛠️ Technologies Used
- PHP
- MySQL (phpMyAdmin)
- XAMPP Server
- Apache Server
- HTML
- CSS

## 📊 Features
- User-friendly interface for booking event spaces
- Store and manage booking details in database
- Check availability of venues
- Admin control for managing bookings

## 🗄️ Database
- Database managed using phpMyAdmin
- Tables include:
  - Users
  - Bookings
  - Event Spaces

## 🚀 How to Run the Project
1. Install XAMPP
2. Start Apache and MySQL in XAMPP
3. Place project folder inside `htdocs`
